## Testen der REST-Schnittstelle

1. Install Dependencies

```bash
$ npm install
```

2. Compiling via Node.js

```bash
$ npm run
```

3. Adding Test-Data or Clear Database

```bash
$ npm run testdata
```

```bash
$ npm run clear
```

4. Access API via a HTTP-Request with Hostadress:3000
